Данная сборка содержит проекты :
1) QWT, взятый с сайта http://sourceforge.net/projects/qwt/?source=typ_redirect;
2)visualizer.

Для усановки и запуска проекта необходимо:
1. Установить в соответствии с документацией Community версию QT 5.4.0 и QTCreator с сайта qt-project.org.
2. Скачать и скомпиллировать проект https://github.com/fominandrey/bnb-simulator в соответствии с приложенной инструкцией.
3. Запустить QTCreator и открыть внутри него main.pro. Затем осуществить сборку проекта.
4. В папку <директория сборки>/visualizer добавить из проекта simulator-bnb файлы build и settings.json.
5. Запустить BnBVisualizer.
